Чтобы протестировать приложение, поместите в папку с ClientTest.exe/ImageTest.exe файл с названием "test.bmp"
(В корне решения лежат изображения, на которых я получил конкретные значения: "test for clients.bmp" и "test for image.bmp" соответственно)
Данные исполняемые файлы на выходе создают "ClientTest.txt"/"ImageTest.txt". 
В корне решения лежат файлы .py, которые по одноимённому текстовому файлу строят графики:
ClientTest.py строит график зависимости среднего времени обработки запроса от количества клиентов
ImageTest.py строит график зависимости времени обработки запроса от размера изображения
Также в корне решения лежат файлы ClientTest.bmp и ImageTest.bmp - результаты тестирования на конкретных 
вышеуказанных исходных изображениях.
